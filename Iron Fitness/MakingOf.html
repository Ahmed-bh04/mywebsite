Making of Iron Fitness

Our Mockup Designs

Introduction

Our Website Iron Fitness is desgined for the people who are intrested in the healthy life and workout in the gym. Our goal for this project is to desgin a user freindly website, which will be helpful for athletes to increase their physical fitness.

Home Page Mockup

The home page of Iron Fitness features a welcoming message, clear navigation, engaging content about healthy living, a user registration form, and greeting, all designed for easy interaction and a personalized experience.

Workout Page Mockup

The Workout page provides an overview of key muscle groups and exercises with embedded YouTube videos. It includes detailed descriptions of exercises like Incline Dumbbell Press, Deadlifts, Lunges, and more.

Food Page Mockup

The Food page offers tips on healthy eating, including food choices, hydration, and exercise. It emphasizes avoiding unhealthy foods, and provides advice on meal planning for long-term wellness.

Store Page Mockup

The Store page displays fitness products like supplements and apparel. Users can add items to the cart, view a summary, and proceed to checkout. JavaScript manages cart actions.

Justification of our design

  • When we incorporate many color and specifically light colors to our web page, but in the Register or Home page we use dark color so it will be better and unique.

  • We used many tables like the table in the Workout page, so there will be many rows and specify the row with different information.

  • We used dark colors for some heading to be clear, and pages background we used light colors for better readability and easy on the user's eyes.

  • On the top right, we made the user name in cyan color to be clear to the user and also give a calm feel to the site.

Justification of our code

  • HTML: We increase the user readability and to keep a good design, so we used the tags like head, nav and footer respectively.

  • CSS: We used guarantees a beautiful styling; and further refinements are very easy to implement.

  • JavaScript: Incorporation of new elements, new objects and new functions was useful to untangle and make the task for the user, easier when browsing this website.

Conclusion

For effective site maintenance, we chose to put all CSS design files in a separate file. We also added a separate page for JavaScript and a separate file for images, which makes it easy to maintain the site in the future. For ease of navigation, we have also added a web navigation bar which makes it easier for the user to navigate through the website pages.
